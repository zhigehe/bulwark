IT之家 2 月 11 日消息，本月初，微软官方网站上出现了一个令人哭笑不得的网页。该网页标题声称将介绍如何卸载微软 Edge 浏览器，然而其内容却大篇幅诋毁竞争对手谷歌 Chrome 浏览器。**这一“假指南”事件引发了大量争议，微软随后不得不将该网页悄然撤下。**

![图片](https://inews.gtimg.com/om_bt/OYARLwdgpfcYgWCzPm2iwZ5C_O1UYahijfGOYvhIzttHUAA/641)

IT之家注意到，如今该链接已指向一个新的网页，**其内容不再伪装成“卸载 Edge 指南”，而是着重推广 Edge 浏览器本身及其功能特性**。例如，网页介绍了 Edge 浏览器的集成 Copilot 功能、垂直标签页、改进的能效表现，以及其他一些真正实用的功能。同时，微软还移除了之前用于比较 Edge 和 Chrome 的表格。

![图片](https://inews.gtimg.com/om_bt/OnLrEfTS_XfN2SNlKiq8h7vJpkLVGsxx0qdLKaL9PmcYMAA/641)

此外，更新后的网页还提供了许多有用的链接，包括 Edge 浏览器的最新功能更新、近期的 Edge 年度回顾总结、数据导入指南、Edge 移动版的下载链接，以及关于必应搜索、AI 主题、无障碍功能、游戏功能、微软奖励计划等更多功能的详细信息。

目前，新页面看起来更像是一个真正有用的资源，帮助用户更好地了解 Edge 浏览器，而不是像之前那样用误导性的标题欺骗用户。不过，这种改变似乎总是发生在公众的强烈反应之后，例如这次事件，还有之前必应搜索伪装成谷歌搜索的事件。

尽管如此，微软仍有一些行为并未因批评而改变。例如，该公司仍在 Chrome 的官方网站上投放广告，试图引导用户放弃使用 Chrome。当用户使用 Edge 浏览器下载谷歌浏览器时，页面顶部仍会显示一条巨大的横幅，声称“微软 Edge 基于与 Chrome 相同的技术”。
