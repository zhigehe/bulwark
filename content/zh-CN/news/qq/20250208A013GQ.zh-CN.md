IT之家 2 月 8 日消息，微软公司今天（2 月 8 日）发布博文，面向 Beta 频道的 Windows Insider 项目成员，发布了适用于 Windows 11 的 KB5052100 更新，**用户安装后版本号升至 Build 22635.4880。**

IT之家援引博文介绍，Windows 11 Build 22635.4880 仅面向没有升级到 Windows 11 24H2 更新的 Beta 频道用户。

一、改进和修复（逐步推出）
-------------

### 1.1、常规

常规更新，改善 Windows 11 预览版整体运行体验。

二、修复（逐步推出）
----------

### 2.1、设置

修复设置应用 Home 页面崩溃的问题。

三、已知问题
------

### 3.1、文件管理器

文件资源管理器在使用 X 按钮关闭时非常慢，也可能影响其他标题栏按钮。

### 3.2、设置

在使用微软账号登录的非管理设备中，出现企业设备信息和可访问性首选项卡。

四、画图应用更新
--------

微软更新画图（Paint）应用（11.2412.271.0 及更高版本），并邀请 Beta 和 Release Preview 频道的 Windows Insider 项目成员测试。

画图应用新增 Copilot 菜单，方便用户访问所有智能功能，包括 Cocreator、Image Creator、Generative erase 和 Remove background，从而更快地完成图像生成和编辑任务。

![图片](https://inews.gtimg.com/om_bt/OHMpzHsmWUm9fZKOuSFWBUo0uXTpavQjUwgJly87anQfYAA/641)

Cocreator 功能目前仅限 Windows 11 AI+ PC 使用，菜单中还包含每个功能的简短描述，方便用户选择合适的工具。
