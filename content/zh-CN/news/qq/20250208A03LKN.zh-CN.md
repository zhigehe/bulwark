IT之家 2 月 8 日消息，科技媒体 WinBuzzer 昨日（2 月 7 日）发布博文，报道称微软更新发布 Visual Studio Code 1.97 版本，**主要在开发流程中整合 GitHub Copilot，提高开发者的开发效率。**

![图片](https://inews.gtimg.com/om_bt/OReyYI9dXATuq4we3LIj7XGMtxhPkmwwbSlPy-r4eLLvUAA/641)

新版整合 GitHub Copilot 后，Copilot 不再是被动地补全代码，而是主动预测开发者下一步可能需要的修改，并给出建议。

IT之家援引博文介绍举例如下，当开发者重命名一个变量时，Copilot 不仅会建议完成当前行的修改，还会识别并建议修改文件中所有该变量的实例。

开发者可以通过设置 `chat.editing.autoAcceptDelay`，无需手动干预，让 Copilot 在一段时间后自动部署建议的修改。

![图片](https://inews.gtimg.com/om_bt/G3GB5n4pqoi4nqfVwpaKWzMs7W0L8uAVApIW8HLTo6V8cAA/0)

VS Code 现在支持 OpenAI 的 o3-mini 和 Google 的 Gemini 2.0 Flash 模型，为开发者提供更多 AI 选择，以满足不同的任务需求。

新版本要求开发者在安装扩展前验证发布者，从而提高安全性，防止恶意扩展入侵开发环境，通过命令行安装扩展将绕过此验证过程。

![图片](https://inews.gtimg.com/om_bt/O2DE-bLkhjzmQcxtFS6yF_xUP9842O-i5pBkCDrSAOaocAA/641)

开发者现在可以自由拖动命令面板到编辑器窗口的任何位置，更好地定制工作区布局。

![图片](https://inews.gtimg.com/om_bt/GcjpSeNxepLR9w0VknPVqMP5S_ZkiE-fsI_9zevcxologAA/0)
