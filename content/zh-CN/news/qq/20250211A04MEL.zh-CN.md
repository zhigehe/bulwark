IT之家 2 月 11 日消息，三星在 One UI 7 Beta 社区发布公告，称目前正准备为 Galaxy S24 系列手机酝酿 One UI 7 Beta 4 更新，**并计划 2 月底或者 3 月初发布 One UI 7 稳定版。**

![图片](https://inews.gtimg.com/om_bt/Ouil73FrPZrAUCoUGyAEDzrO4Lwm9r1fwpTlbZ4k0wnmcAA/641)

IT之家此前报道，三星正进一步打磨 One UI 7 系统，消息称 One UI 7 第 4 个 Beta 更新将着重解决屏幕偏色问题。此前，部分用户反馈在 One UI 7 的开发版本中，当屏幕色彩模式设置为“自然模式（Natural Mode）”时，会出现屏幕泛黄的现象。三星建议遇到此问题的测试用户，暂时将色彩模式调整为“正常（Normal）”模式。

据悉，One UI 7 将会带来诸多基于人工智能 (AI) 的全新功能。虽然这些 AI 功能可能与未来的 S25 系列相比存在一定限制，但开发团队正在努力优化，力求在稳定版发布时提供最佳体验。稳定版的 One UI 7 势必会给用户带来更加智能和便捷的使用体验。
